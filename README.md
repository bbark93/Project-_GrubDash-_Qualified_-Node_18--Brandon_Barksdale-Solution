
# GrubDash

This is an meal order and delivery app used to practice route handlers and error handlers

## AI Usage
generative AI was not used but AI was used as one method of troubleshooting. An example would be AI letting me know "const newId = nextId;" assigns the function to newId but does not invoke the function and that I would need to add () to the end of nextId to invoke the function and assign it's value to newId. The reason I did this was due to technical issues deploying a server.
